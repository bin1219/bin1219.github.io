keil5快捷键

代码文本编辑

把选中的行注释：Ctrl/Command +
把选中的行取消注释：Ctrl/Command + /
把当前行的内容剪切：Ctrl/Command + L
跳转到指定行：Ctrl/Command + G

复制选中内容：Ctrl/Command + C
粘贴选中内容：Ctrl/Command + V
剪切选中内容：Ctrl/Command + X
查找：Ctrl/Command + F
替换：Ctrl/Command + H
重做：Ctrl/Command + Y
撤销：Ctrl/Command + Z
全选：Ctrl/Command + A

缩进：Tab
取消缩进：Shift + Tab

移动光标选中： Shift + 方向键

文件编辑

创建新文件：Ctrl/Command + N
打开文件：Ctrl/Command + O
保存文件：Ctrl/Command + S

编译

运行程序：F5
代码下载：F8
创建目标文件：F7
